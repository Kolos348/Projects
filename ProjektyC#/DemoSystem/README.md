# DemoSystem - Animation Demo Project

--ENGLISH README--

1. Title - Animation Demo Project

2. Description:
This project is an example demo consisting of a series of 4 animated elements:
- A sinusoid that shrinks and expands
- Many circles of different sizes bouncing off each other
- Three triangles penetrating through each other up and down
- A wall that rises and falls

This is a practical demonstration of animation and visual effects in C# using .NET SDK version 10.

--POLISH README--

3. Tytuł - Projekt Animacji Demo

4. Opis projektu:
Projekt jest przykładem dema (seria animacji) składające się z 4 elementów:
- kurczącej i rozszerzającej się sinusoidy,
- dużej ilości kół o różnych rozmiarach odbijających się od siebie,
- trzech trójkątach przenikających przez siebie w górę i w dół,
- wali, która unosi się i opada.

Jest to praktyczna demonstracja animacji i efektów wizualnych w C# z użyciem .NET SDK version 10.
