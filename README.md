# Projects - Portfolio Repository

--ENGLISH README--

1. Overview:
This is a comprehensive portfolio repository containing various C# projects and web applications. The repository demonstrates software development skills across multiple domains including design patterns, web development, and AI integration.

All projects are organized into two main categories: C# desktop/console applications and web-based projects.

2. Prerequisites:
Before cloning and running projects from this repository, ensure you have the following installed:

**For C# Projects:**
- .NET SDK version 10 or later
- Visual Studio 2022 or Visual Studio Code
- Windows operating system (for some projects)

**For Web Projects:**
- Node.js (version 16 or later)
- npm (Node Package Manager)
- A modern web browser
- OpenSSL (for generating SSL certificates)

3. Project Categories:

**ProjektyC# (C# Projects)**
- DemoSystem – An animated demo application showcasing visual effects and animations
- Wzorce Projektowe (Design Patterns) – Examples of Facade and Composite design patterns

**Witryny Internetowe (Web Projects)**
- ChatbotAI – An AI-powered chatbot integrated with Google Gemini API
- Statyczne – Static website showcasing HTML and CSS skills
- Dynamiczne/Techzone – Dynamic web application with backend integration

4. Getting Started:

**For C# Projects:**
1. Navigate to the ProjektyC# folder
2. Open the desired project folder
3. Read the project-specific README.md
4. Build using Visual Studio or: `dotnet build`
5. Run the application

**For Web Projects:**
1. Navigate to the Witryny Internetowe folder
2. Choose your project (ChatbotAI, CoffeShop, Techzone, etc.)
3. Read the project-specific README.md for setup instructions
4. Install dependencies: `npm install`
5. Create required configuration files (.env, SSL certificates)
6. Run the project according to its README instructions

5. Technologies Used:
- **Languages:** C#, JavaScript, Python, HTML, CSS
- **Frameworks:** .NET, Express.js, Socket.io
- **Tools:** Visual Studio, Node.js, PM2, Google Gemini API
- **Patterns:** Design Patterns, MVC, REST API

6. Repository Structure:

```
C-Projects/
├── ProjektyC#/
│   ├── DemoSystem/           (Animation demonstration)
│   ├── Wzorce Projektowe/
│   │   ├── Fasda/           (Facade pattern)
│   │   └── Kompozyt/        (Composite pattern)
│   └── README.md
├── Witryny Internetowe/
│   ├── ChatbotAI/           (AI chatbot service)

│   ├── Dynamiczne/          (Dynamic applications)
│   └── README.md
└── README.md                (This file)
```

7. Setup & Configuration:

Each project has specific setup requirements. Always refer to the individual project README.md files for:
- Environment variables (.env file examples)
- Dependency installation commands
- Build instructions
- How to run the application
- Port configuration (for web projects)
- Certificate generation (for HTTPS projects)

8. Author & Contact:
This repository represents a portfolio of work demonstrating proficiency in multiple programming languages and software development practices.

For more information about specific projects, refer to their individual README.md files.
